# Sales Analytics with SQL and Python

This project performs a comprehensive sales analysis using SQL and Python on an e-commerce transactions dataset. It demonstrates how to extract actionable business insights, compute key metrics, and understand customer and product performance.

---

## Dataset

The dataset (`sales.db`) is not included due to size. You can use a sample CSV or connect to your own e-commerce database to replicate the analysis. Ensure the database has the same schema as the original.

---

## üîç Project Overview

The analysis covers:

- Total revenue generated by each customer
- Identification of the top 10 customers by spend
- Top revenue-generating products
- Monthly revenue trends
- Repeat customer and purchase frequency analysis
- Average Order Value (AOV)
- Revenue distribution by country
- Sales volume by time of day
- Identification of customers at risk of churn
- Revenue contribution of the top 20% of customers

The project uses **SQLite** for SQL-based querying and **Python (pandas)** for data handling and analysis.
